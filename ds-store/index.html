.DS_Store Parser

Extract hidden filenames and directory listings from macOS .DS_Store binary files. Useful for web server reconnaissance.

📂 Upload .DS_Store File

What is .DS_Store? macOS creates .DS_Store files in every folder to store Finder metadata. When misconfigured web servers expose them, attackers can enumerate the directory structure. This tool parses the binary format entirely in-browser — no data is uploaded.
